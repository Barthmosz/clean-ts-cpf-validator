# Clean Cpf Validator

## Installation
---
```npm i clean-cpf-validator```

<br/>

## Usage
---
#### JavaScript

```javascript
const { CpfValidator } = require('clean-cpf-validator')

const isValid = CpfValidator.validate('123.456.789-12') // true
```

#### TypeScript
```typescript
import { CpfValidator } from 'clean-cpf-validator'

const isValid = CpfValidator.validate('123.456.789-12') // true
```
